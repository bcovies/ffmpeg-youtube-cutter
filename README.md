# ffmpeg-youtube-cutter

Container para cortar vídeos do youtube e disponibilizar o download do mesmo para o usuário.
# Como usar?

Navege até a pasta compose e execute: 
```
docker-compose up -d
```

Abra o link no seu navegador referente ao container. Exemplo: ``` localhost:1000/index.html``` e preencha os campos desejados como no vídeo a seguir:
